# vwの計算
- １vwは画面横幅の100分の1になる
>例)<br>
>画面横幅が３２０pxの時にh１要素を４８pxで表示したい時<br>
>1vw = 3.2px<br>
>48 % 3.2 = 15 になるから<br>
>48px = 15vw

# emの計算
- フォントサイズ = 1em
>例）<br>
>font-sizeが115pxだとすると<br>
>1em = 115px<br>
>0.1em = 11.5px<br>
>0.2em = 23pxとなる
